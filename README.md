# Sample Markdown Project

This is a sample project to demonstrate Markdown to PDF conversion using Python.

## Features

- **Easy to use**: Simple command line interface.
- **Fast**: High-performance conversion.
- **Customizable**: Supports CSS and layout adjustments.

## Code Example

```python
def hello_world():
    print("Hello, PDF!")
```

## Requirements

1. Python 3.12+
2. `uv` for dependency management
3. `markdown-pdf` library

## Usage

To convert a Markdown file to PDF, use the following command:

```bash
# Если uv добавлен в PATH:
uv run markdowntopdf.py README.md

# Если uv не добавлен в PATH (Windows):
py -m uv run markdowntopdf.py README.md
```

This will create a `README.pdf` in the same directory.

You can also specify an output path:

```bash
py -m uv run markdowntopdf.py README.md -o output.pdf
```
